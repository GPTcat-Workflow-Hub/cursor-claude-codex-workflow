# Claude 出了个"不写代码的 Claude Code"，要抢你的办公桌

更新时间：2026/07/08 · 账号 B · 资讯向

Anthropic 把 Claude Cowork 扩展到了手机和网页。这个被称为"没有代码的 Claude Code"的工具，标志着 AI agent 正从程序员的电脑，蔓延到每个人的办公桌。

---

## Claude Cowork 是什么

一句话：**Claude Code 的能力，搬给不写代码的普通白领。**

Claude Code 是帮程序员干活的 agent——读代码、改文件、跑测试。Cowork 把同一套"自主执行任务"的能力，用到了办公场景：写报告、做表格、整理数据、做 PPT。

它 2026 年 1 月先出了桌面版。7 月 7 日起，**扩展到网页和手机**（限 Max 订阅用户）。关键升级是跨设备：

- 在电脑上发起一个任务
- 用手机看进度
- 合上笔记本，任务在后台继续跑
- 完成后再回来取结果

Anthropic 想让它像个"agent 行政同事"，而不是一个编程工具。

---

## 一组数据，暴露了真相

Anthropic 公布了一份用量数据（120 万匿名会话，来自 60 万+ 组织，5 月最后两周）：

| 用途 | 占比 |
|------|------|
| 业务流程操作（报告、入职清单、对账表）| 33.4% |
| 内容创作/文案（草稿、PPT、社媒）| 16.4% |
| **软件开发** | **仅 8.7%** |

看清楚最后一行：**软件开发只占 8.7%。**

这说明一件事：这些"编程 agent"，早就不只是程序员在用了。绝大多数用量来自财务、HR、行政这些**根本不写代码**的岗位。他们用 AI 处理的是"工作里那些琐碎的事"。

---

## 为什么大厂都在往这个方向走

不只是 Anthropic。OpenAI 的 Codex 也走了同一条路——从开发工具起家，现在服务非开发者做报告、表格、PPT、研究、分析。

背后的逻辑，TechCrunch 点得很透：**成败"不再取决于谁有最好的聊天机器人，而取决于谁占住了工作真正发生的地方"。**

聊天机器人是"你问它答"，你还得自己动手。agent 是"你派活它干"，直接出结果。谁能让 agent 无缝嵌进你的日常办公，谁就锁住了用户。

Anthropic 最近还推了 Claude Tag——常驻在 Slack 里的 Claude。方向一致：让 AI 待在你已经在用的工作场所里。

---

## 对普通人的意义

这个趋势跟你有关，哪怕你不写代码：

1. **AI 正在从"聊天"变成"干活"**。以后评价一个 AI 工具，不看它能聊多好，看它能不能真的替你完成任务。
2. **办公自动化的门槛在降低**。以前要写脚本、学工具才能自动化的活，现在用大白话派给 agent 就行。
3. **会用 agent 会成为新的办公技能**。就像当年会用 Excel 一样。
